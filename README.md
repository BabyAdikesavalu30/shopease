# ShopEase 🛍️

A modern men's e-commerce website built with React.

## Features
- 32+ men's products across 5 categories
- Filter by category and sort by price
- Live search products by name
- Add to cart with quantity controls
- Toast notifications
- Checkout with form validation
- Cart saved to localStorage
- Fully responsive on mobile

## Tech Stack
- React
- React Router
- Context API
- CSS3
- localStorage
